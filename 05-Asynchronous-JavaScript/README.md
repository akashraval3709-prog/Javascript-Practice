# ⚡ Asynchronous JavaScript

A collection of JavaScript examples demonstrating asynchronous programming using **Promises**, **async/await**, and **Promise.all()**. These examples simulate real-world API requests and teach modern asynchronous JavaScript concepts.

## 📚 Topics Covered

- Promises
- `resolve()`
- `reject()`
- `async`
- `await`
- `try...catch`
- `finally()`
- `Promise.all()`
- Parallel Data Fetching
- Error Handling
- `setTimeout()`

## 📄 Examples

### 02_async_await_user_profile.js

- Simulates fetching a user profile
- Uses Promises with `async/await`
- Demonstrates `try...catch...finally`

### 03_movie_data_fetch.js

- Simulates fetching movie information
- Uses Promise-based asynchronous programming
- Handles success and error scenarios

### 04_parallel_data_fetch.js

- Demonstrates `Promise.all()`
- Fetches user, restaurants, and coupons in parallel
- Shows efficient handling of multiple asynchronous tasks

---

## 🎯 Goal

To understand asynchronous programming in JavaScript by practicing Promises, `async/await`, error handling, and parallel execution of asynchronous operations.
