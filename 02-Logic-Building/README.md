# Logic Building

This section focuses on improving problem-solving skills using JavaScript.

---

## Topics Covered

- Conditionals
- Loops
- Functions
- Nested Loops
- Patterns
- Arrays Basics
- Switch Statements
  
---

## Practice Goals

- Build strong programming logic
- Solve coding problems
- Improve thinking approach
- Prepare for real-world projects

---

## Files

### 01_temperature_checker.js
A simple JavaScript program that checks temperature conditions and displays:
- Hot Weather
- Nice Weather
- Cold Weather

Concepts used:
- if-else statements
- Comparison operators
- Conditional logic

---

### 02_month_switch_case.js
Displays month names based on month numbers using switch-case statements.

Concepts used:
- switch-case
- break statement
- Default case handling

---

### 03_data_type_checker.js
Checks whether a variable is a string, number, or another data type using the typeof operator.

Concepts used:
- switch-case
- typeof operator
- Data type checking

---

### 04_password_strength_checker.js
A password validation program that checks password strength based on:
- Uppercase letters
- Lowercase letters
- Numbers
- Password length

Password strength levels:
- Weak Password
- Medium Password
- Strong Password

Concepts used:
- Functions
- Loops
- Conditions
- String checking
- Boolean logic
  
---

### 05_function_multiply.js

A simple JavaScript function that multiplies two numbers and returns the result.

Concepts used:
- Functions
- Parameters
- Return keyword
- Function hoisting
