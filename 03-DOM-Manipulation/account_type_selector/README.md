# 🏦 Account Type Selector

A simple JavaScript DOM Manipulation project that displays account information based on the selected account type. Users can choose an account from a dropdown menu and instantly view the corresponding interest rate details.

## ⚙️ How It Works

1. User selects an account type from the dropdown menu.
2. JavaScript listens for the change event.
3. Based on the selected option, account information is displayed.
4. The message updates dynamically without reloading the page.
