# Color Scheme Switcher

A simple JavaScript DOM Manipulation project that changes the webpage background color when users click on different color buttons.


## 🚀 Live Demo
You can view the live project here:  
👉 [Live Project Link](https://akashraval3709-prog.github.io/Javascript-Practice/03-DOM-Manipulation/01_color_scheme_switcher/)

## Features

- Change background color dynamically
- Update text color based on selected background
- Interactive button click events
- Beginner-friendly DOM project

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Files

### index.html
Contains:
- Page structure
- Color buttons
- Styling for buttons and layout
- JavaScript file linking

### ColorSchemeSwitcher.js
Handles:
- DOM element selection
- Button click events
- Background color changes
- Dynamic text color updates

## Concepts Practiced

- querySelector()
- querySelectorAll()
- addEventListener()
- forEach()
- Event Object
- event.target
- DOM Manipulation
- Dynamic Styling
- CSS Properties with JavaScript

## How It Works

1. User clicks a color button.
2. JavaScript detects the clicked button.
3. The page background changes to the button color.
4. Text color automatically updates for better visibility.
