# 🔐 OTP Verification System

A simple JavaScript project that generates a random 4-digit OTP and verifies user input. The application includes an OTP expiration timer, resend functionality, and real-time verification feedback.

## ⚙️ How It Works

1. Click the **Get OTP** button.
2. A random 4-digit OTP is generated.
3. The OTP remains valid for 30 seconds.
4. Enter the OTP in the input field.
5. If the OTP matches, verification succeeds.
6. If the OTP is incorrect or expires, the user can request a new OTP.
