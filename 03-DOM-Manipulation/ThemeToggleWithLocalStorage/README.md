# 🌙 Theme Toggle With Local Storage

A simple JavaScript project that allows users to switch between Light Mode and Dark Mode. The selected theme is saved using Local Storage so that the user's preference remains available after refreshing the page.
 🚀 Live Demo

## 🚀 Features

* Toggle between Light and Dark themes
* Save theme preference using Local Storage
* Dynamic style updates

## ⚙️ How It Works

1. The application checks the saved theme from Local Storage.
2. If a theme exists, it is applied automatically.
3. When the user clicks the toggle button, the theme changes.
4. The selected theme is stored in Local Storage.
5. The preference remains available after page refresh.
