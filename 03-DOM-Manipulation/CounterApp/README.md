# 🔢 Counter App

A simple JavaScript Counter Application that allows users to increase, decrease, and reset a counter value using button clicks. The application also saves the counter value using Local Storage, ensuring data persistence across page reloads.

## ✨ Features

* Increase counter value
* Decrease counter value
* Reset counter to zero
* Prevent negative values
* Save counter value using Local Storage
* Restore counter value after page reload
* Dynamic color feedback for actions

## ⚙️ How It Works

1. Click **Increase (+)** to increment the counter.
2. Click **Decrease (-)** to decrement the counter.
3. Click **Reset** to set the counter back to zero.
4. Counter values are automatically saved in Local Storage.
5. The saved value is restored when the page reloads.
6. The counter color changes based on the performed action.
