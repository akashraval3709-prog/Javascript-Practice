🌐 03-DOM Manipulation
========================

This section is about interacting with web pages using JavaScript.

📌 Topics
---------
- Selecting elements (getElementById, querySelector)
- Changing content (innerText, innerHTML)
- Styling elements using JS
- Events (click, input, submit)
- Creating & removing elements dynamically

🎯 Goal
-------
Make webpages interactive and dynamic using JavaScript.

🚀 Overall Goal
--------------
To become confident in JavaScript by moving step-by-step from basics → logic building → real-world DOM manipulation.
