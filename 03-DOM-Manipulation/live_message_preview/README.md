# ✍️ Live Message Preview

A simple JavaScript DOM Manipulation project that displays user input in real time. As the user types a message, the text is instantly converted to uppercase and displayed on the screen.

## ⚙️ How It Works

1. User enters text into the input field.
2. JavaScript listens for the input event.
3. The entered text is converted to uppercase.
4. The transformed text is displayed instantly on the page.
