# 🕒 Digital Clock

A simple and interactive Digital Clock built using HTML, CSS, and JavaScript. The clock displays the current local time and updates automatically every second in real time.

## 🎯 Project Goal

To practice JavaScript DOM Manipulation and learn how to work with the Date API by building a real-time digital clock.

---

## ⚙️ How It Works

1. JavaScript selects the clock element from the DOM.
2. A new Date object is created every second.
3. The current time is converted into a readable format.
4. The clock display updates automatically using setInterval().
