# 🌐 Fetch API

A collection of JavaScript examples demonstrating how to use the Fetch API to communicate with REST APIs using modern asynchronous JavaScript.

## 📚 Topics Covered

- Fetch API
- GET Requests
- Response Object
- JSON Parsing
- async / await
- Promise
- try...catch
- finally
- HTTP Error Handling

---

## 📄 Examples

### 01_fetch_dashboard_data.js

- Fetches dashboard data from a REST API
- Uses Fetch API with `async/await`
- Validates HTTP response status
- Parses JSON data
- Handles errors using `try...catch`
- Executes cleanup using `finally`
