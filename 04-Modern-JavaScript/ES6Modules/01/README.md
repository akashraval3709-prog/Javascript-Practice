# 📦 ES6 Modules - Example 01

This example demonstrates the basics of **ES6 Modules** by exporting variables and functions from one module and importing them into another.

---

## 🚀 Concepts Covered

- Named Export
- Default Export
- Named Import
- Default Import
- Reusable Modules
- ES6 Module Syntax

---

## 📂 Files

### cyberSecurity.js

Contains:

- Secret key
- Named export
- Default export
- `encryptData()` function
- `decryptData()` function

### main.js

Handles:

- Importing named exports
- Importing default export
- Calling exported functions
- Displaying module output in the console

---

## 📚 Learning Concepts

- `export`
- `export default`
- `import`
- Named Imports
- Default Imports
- Modular Programming
- Code Reusability

---

## 🎯 Goal

To understand how ES6 Modules allow JavaScript code to be split into reusable files using `export` and `import`.
