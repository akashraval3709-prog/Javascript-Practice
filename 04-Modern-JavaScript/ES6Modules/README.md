# 📦 ES6 Modules

A simple JavaScript project demonstrating how to use **ES6 Modules** with `export`, `named exports`, `default export`, and `import` statements to organize reusable code.

---

## 🚀 Features

- Export variables and functions
- Default export
- Named exports
- Import modules
- Reusable JavaScript code
- Beginner-friendly ES6 Modules example

---

## 🛠️ Technologies Used

- JavaScript (ES6 Modules)

---

## 📚 Learning Concepts

- export
- export default
- import
- Named Export
- Default Export
- Modular Programming
- Code Reusability

---

## 📂 Project Structure

```text
ES6Modules/
│
├── cyberSecurity.js
├── main.js
└── README.md
```

---

## 📄 Files

### cyberSecurity.js

Contains:

- Secret key
- Named exports
- Default export
- Encryption function
- Decryption function

### main.js

Handles:

- Importing modules
- Using named imports
- Using default imports
- Calling exported functions
- Displaying results in the console

---

## 🎯 Project Goal

To understand how ES6 Modules organize JavaScript code into reusable and maintainable files using `export` and `import`.
