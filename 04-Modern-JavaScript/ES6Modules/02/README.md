# 📦 ES6 Modules - Example 02

This example demonstrates how to organize JavaScript code into reusable modules by separating GST calculation logic from product pricing logic using **ES6 Modules**.

---

## 🚀 Concepts Covered

- Named Export
- Named Import
- Reusable Functions
- Module Organization
- ES6 Module Syntax

---

## 📂 Files

### taxCalculator.js

Contains:

- `calculateGST()` function
- Named export
- GST calculation logic

### productStore.js

Handles:

- Importing the GST module
- Calculating the final product price
- Displaying the result in the console

---

## 📚 Learning Concepts

- `export`
- `import`
- Named Exports
- Named Imports
- Modular Programming
- Function Reusability
- Code Organization

---

## 🎯 Goal

To understand how ES6 Modules improve code organization by separating business logic into reusable JavaScript files.
